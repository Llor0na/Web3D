# WebGL
Ce projet explore la possibilité d'extraire des données 3D à partir de sources web, telles que Google Maps, Sketchfab et Marmoset Viewer. 

La règle essentielle est que si vous pouvez voir ces objets sur votre ordinateur, vous pouvez également les enregistrer, un peu comme prendre une photo.

Cela fonctionne de la même manière que lorsque vous trouvez des images en ligne, mais avec des objets 3D. 
Cependant, chaque site web a sa propre façon de stocker ces objets 3D, ce qui rend nécessaire la compréhension de leurs règles spécifiques.

Ce projet sert de base pour explorer davantage la récupération de données 3D depuis le web, en gardant à l'esprit les spécificités de chaque source de données.
